# Obsidian Git Setup

## Recommended Settings

- Auto pull on startup.
- Auto commit and sync on a calm interval.
- Use meaningful commit messages when possible.
- Prefer desktop for heavier Git operations.
- Avoid running multiple sync systems against the same vault.

## Agent-Generated Changes

Codex and Jules should work through branches and pull requests. Do not silently rewrite `main` with generated changes.

## Mobile Note

Do not assume mobile Git is reliable for every workflow. Use mobile for capture and desktop for review-heavy changes.

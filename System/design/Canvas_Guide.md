# Canvas Guide

> The canvas is the visual control map for the Life OS.

## How to Use It

Open [[Maps/Life_OS_Command_Center.canvas]] when you want the vault to feel like a command center instead of a folder tree.

## What It Shows

- Life OS Home at the center.
- Capture and daily notes on the left.
- Tasks and active projects on the right.
- Domain dashboards below.
- Weekly reviews and system reports as feedback loops.

## Maintenance

Keep the canvas simple. Add only high-level dashboards, not every note. If a domain grows, create a domain-specific canvas later.

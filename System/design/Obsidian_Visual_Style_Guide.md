# Obsidian Visual Style Guide

## Theme

Use Minimal Theme with Minimal Theme Settings and Style Settings. Keep the vault calm, dense, and readable.

## CSS Snippet

Enable `.obsidian/snippets/life-os.css` in Obsidian settings under Appearance > CSS snippets.

## Dashboard Pattern

Every dashboard should answer:

- Where am I?
- What needs attention?
- What can I do next?
- What needs review?

## Tables

Use small tables for navigation and comparison. Avoid giant tables for emotional or reflective material.

## Callouts

Use callouts for warnings, review notes, and important operating principles. Do not turn every paragraph into a callout.

## Readability

Prefer short sections, clear links, and query blocks with static links nearby. The vault should remain useful even when plugins are disabled.

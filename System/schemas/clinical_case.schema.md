# Clinical Case Schema

## Required Frontmatter

```yaml
type: clinical_case
deidentified: true
review_needed: true
```

## Required Headings

- `# Case`
- `## Presentation`
- `## Findings`
- `## Assessment`
- `## Learning Points`

## Optional Headings

- `## Red Flags`
- `## Follow Up`

## Example

```markdown
---
type: clinical_case
deidentified: true
review_needed: true
---
# Deidentified Case

## Presentation

## Findings

## Assessment

## Learning Points
```

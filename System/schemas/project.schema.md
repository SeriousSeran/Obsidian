# Project Schema

## Required Frontmatter

```yaml
type: project
status:
review_needed: false
```

## Required Headings

- `# Project`
- `## Outcome`
- `## Next Actions`
- `## Decisions`
- `## Review`

## Optional Headings

- `## Milestones`
- `## Risks`

## Example

```markdown
---
type: project
status: active
review_needed: false
---
# Life OS Engine

## Outcome

## Next Actions

## Decisions

## Review
```

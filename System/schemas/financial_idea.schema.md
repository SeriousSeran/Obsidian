# Financial Idea Schema

## Required Frontmatter

```yaml
type: financial_idea
review_needed: true
```

## Required Headings

- `# Idea`
- `## Thesis`
- `## Assumptions`
- `## Risks`
- `## Next Review`

## Optional Headings

- `## Cash Flow`
- `## Experiments`

## Example

```markdown
---
type: financial_idea
review_needed: true
---
# Subscription Product Idea

## Thesis

## Assumptions

## Risks

## Next Review
```

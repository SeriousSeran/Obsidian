# Medical Topic Schema

## Required Frontmatter

```yaml
type: medical_topic
review_needed: true
deidentified: true
```

## Required Headings

- `# Topic`
- `## Core Concept`
- `## Clinical Pattern`
- `## Questions`
- `## Review`

## Optional Headings

- `## Differential`
- `## OSCE/Viva`

## Example

```markdown
---
type: medical_topic
review_needed: true
deidentified: true
---
# Chest Pain

## Core Concept

## Clinical Pattern

## Questions

## Review
```

# Content Idea Schema

## Required Frontmatter

```yaml
type: content_idea
status:
review_needed: false
```

## Required Headings

- `# Content Idea`
- `## Hook`
- `## Core Point`
- `## Audience`
- `## Next Action`

## Optional Headings

- `## Outline`
- `## Publish Notes`

## Example

```markdown
---
type: content_idea
status: draft
review_needed: false
---
# Why Medical Students Need Better OSCE Tools

## Hook

## Core Point

## Audience

## Next Action
```

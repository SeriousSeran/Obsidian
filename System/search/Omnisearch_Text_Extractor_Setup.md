# Omnisearch and Text Extractor Setup

## Use Cases

- Find old viva notes.
- Search PDFs and screenshots.
- Search medical terms.
- Find deidentified patient-learning notes.
- Find product and content ideas.

## Guidance

Use Omnisearch for fast retrieval across the vault. Use Text Extractor for manual OCR of screenshots and PDFs. The Life OS scripts do not rely on OCR, so search improvements never become a hidden automation dependency.

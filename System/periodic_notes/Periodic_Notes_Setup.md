# Periodic Notes Setup

| Period | Folder | Format | Template |
|---|---|---|---|
| Daily | `Daily/Daily_Notes` | `YYYY-MM-DD` | `Templates/Daily_Note_Template.md` |
| Weekly | `Daily/Weekly_Reviews` | `YYYY-[W]WW` | `Templates/Weekly_Review_Template.md` |
| Monthly | `Daily/Monthly_Reviews` | `YYYY-MM` | `Templates/Monthly_Review_Template.md` |
| Quarterly | `Daily/Quarterly_Reviews` | `YYYY-[Q]Q` | `Templates/Quarterly_Review_Template.md` |

Use Calendar for daily navigation and Periodic Notes for review creation. The CLI can also create daily and weekly skeletons.

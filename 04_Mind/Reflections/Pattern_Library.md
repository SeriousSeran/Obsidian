# Pattern Library

> Repeating personal patterns, tracked gently without diagnosis.

## Pattern Table

| Pattern | Trigger | Helpful Response | Review |
|---|---|---|---|
| Avoidance loop | Unclear next action | Define one small action | Weekly |
| Overbuild loop | Exciting system idea | Ship a tiny working version | Weekly |
| Rumination loop | Open emotional question | Write facts, feelings, needs | Weekly |

## Reflection Rules

- Describe behaviour before interpreting it.
- Do not diagnose yourself from a note.
- Look for repeated evidence across time.
- Convert insight into one next action.

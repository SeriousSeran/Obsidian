# Content Operating System

> Turn raw ideas into useful teaching, scripts, posts, and products.

## Pipeline

| Stage | What Happens |
|---|---|
| Capture | Raw idea goes to [[Content/Ideas]] |
| Clarify | Define audience, pain, promise |
| Draft | Write the simplest useful version |
| Edit | Remove vagueness and private details |
| Publish | Only after privacy review |
| Repurpose | Turn strong ideas into multiple formats |

## Content Pillars

- Medicine learning
- AI-assisted systems
- Student productivity
- Money and business experiments
- Reflection and personal operating systems

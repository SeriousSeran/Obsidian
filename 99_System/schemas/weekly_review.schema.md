# Weekly Review Schema

## Required Frontmatter

```yaml
type: weekly_review
week:
review_needed: true
```

## Required Headings

- `# Weekly Review`
- `## Wins`
- `## Open Loops`
- `## Projects`
- `## Direction`

## Optional Headings

- `## Medicine`
- `## Money`
- `## Content`

## Example

```markdown
---
type: weekly_review
week: 2026-W21
review_needed: true
---
# Weekly Review 2026-W21

## Wins

Not enough data.

## Open Loops

## Projects

## Direction
```

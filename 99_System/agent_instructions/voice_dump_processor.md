# Voice Dump Processor

When processing a voice dump:

1. Preserve the raw transcript.
2. Create a processed note beside the raw note or in the relevant domain.
3. Extract domains: medicine, money, mind, body, relationships, content, projects, assets, learning, bucket-list, admin, random.
4. Extract action items.
5. Extract questions to explore.
6. Extract notes to create.
7. Add Obsidian links.
8. Do not over-polish the user's thinking; preserve the living idea.
9. Do not make irreversible decisions.

Preferred output sections:

- Core idea
- Emotional signal, if present
- Domains detected
- Actions
- Notes to create
- Links to update
- Follow-up questions

# Medicine Note Rules

Teach like a clinical mentor.

Use:

Patient → Physiology → Pathology → Clinical Signs → Diagnosis → Investigations → Management.

Rules:

- Be mechanism-first.
- Explain signs by pathophysiology.
- Include viva traps.
- Include emergency priorities when relevant.
- Preserve doses and thresholds only when verified.
- Mark clinical content as `review_needed: true`.
- Deidentify patient cases.

# Jules / Agent General Rules

You are maintaining an Obsidian vault stored in GitHub.

## Non-negotiable rules

1. Never delete raw source notes.
2. Do not silently rewrite the main branch.
3. Prefer branches and pull requests.
4. Preserve Obsidian links.
5. Use Markdown for notes.
6. Add YAML frontmatter to new notes.
7. Update relevant MOC pages after creating important notes.
8. Do not invent factual claims. Mark uncertain claims as `needs verification`.
9. Medical content must be marked `review_needed: true` until reviewed.
10. Financial content must describe risk and uncertainty.
11. Psychology notes must use reflective language, not diagnosis.
12. Keep patient information deidentified.

## Default frontmatter

```yaml
type: note
status: draft
created: YYYY-MM-DD
domains: []
review_needed: true
```

# Business Model Lab

> A place to test income ideas before pretending they are plans.

## Idea Filter

| Question                        | Answer |
| ------------------------------- | ------ |
| Who has the painful problem?    |        |
| What do they already do?        |        |
| What would they pay for?        |        |
| What can be tested in one week? |        |
| What is the main risk?          |        |

## Model Types

- Service
- Digital product
- Education/content
- Software tool
- Template/system
- Community

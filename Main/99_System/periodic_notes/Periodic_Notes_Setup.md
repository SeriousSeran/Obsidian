# Periodic Notes Setup

| Period | Folder | Format | Template |
|---|---|---|---|
| Daily | `01_Daily/Daily_Notes` | `YYYY-MM-DD` | `90_Templates/Daily_Note_Template.md` |
| Weekly | `01_Daily/Weekly_Reviews` | `YYYY-[W]WW` | `90_Templates/Weekly_Review_Template.md` |
| Monthly | `01_Daily/Monthly_Reviews` | `YYYY-MM` | `90_Templates/Monthly_Review_Template.md` |
| Quarterly | `01_Daily/Quarterly_Reviews` | `YYYY-[Q]Q` | `90_Templates/Quarterly_Review_Template.md` |

Use Calendar for daily navigation and Periodic Notes for review creation. The CLI can also create daily and weekly skeletons.

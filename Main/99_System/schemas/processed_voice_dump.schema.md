# Processed Voice Dump Schema

## Required Frontmatter

```yaml
type: processed_voice_dump
source:
review_needed: true
```

## Required Headings

- `# Title`
- `## Source`
- `## Core Insight`
- `## Action Items`
- `## Links`

## Optional Headings

- `## Project Updates`
- `## Review Notes`

## Example

```markdown
---
type: processed_voice_dump
source: 00_Inbox/Voice_Dumps/Recording.md
review_needed: true
---
# Processed Recording

## Source

Raw transcript link.

## Core Insight

The useful signal.

## Action Items

- [ ] One next action.

## Links
```

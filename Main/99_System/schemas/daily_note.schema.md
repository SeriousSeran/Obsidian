# Daily Note Schema

## Required Frontmatter

```yaml
type: daily_note
date:
review_needed: false
```

## Required Headings

- `# Date`
- `## Capture`
- `## Priorities`
- `## Actions`
- `## End of Day Review`

## Optional Headings

- `## Medicine`
- `## Money`
- `## Body`

## Example

```markdown
---
type: daily_note
date: 2026-05-21
review_needed: false
---
# 2026-05-21

## Capture

## Priorities

## Actions

## End of Day Review
```

# Obsidian Plugin Setup

## Essential

| Plugin | Why Used | Vault Usage | Important Settings | If Disabled |
|---|---|---|---|---|
| Dataview | Live dashboards | Review queues, recent notes | Enable JavaScript only if you later need it | Static links still work |
| Tasks | Action tracking | Task Dashboard and domain task blocks | Use due/scheduled dates | Markdown tasks remain visible |
| Templater | Better templates | Optional date and filename automation | Template folder: `90_Templates` | Templates still work as plain Markdown |
| QuickAdd | Fast capture | Raw ideas, voice dumps, projects | See [[99_System/quickadd/QuickAdd_Setup]] | Use templates manually |
| Periodic Notes | Review rhythm | Daily, weekly, monthly, quarterly notes | See [[99_System/periodic_notes/Periodic_Notes_Setup]] | CLI can create daily/weekly notes |
| Calendar | Daily navigation | Calendar view for daily notes | Link to Daily Notes folder | Folder links still work |
| Homepage | Calm startup | Opens [[10_Maps_Of_Content/Life_OS_Home]] | Homepage file path in setup guide | Open home manually |

## Visual / Quality

| Plugin | Why Used | Vault Usage | Important Settings | If Disabled |
|---|---|---|---|---|
| Minimal Theme | Calm visual base | Dashboards and reading | Use readable line width | Default theme still works |
| Minimal Theme Settings | Fine control | Dashboard polish | Keep contrast comfortable | CSS still applies lightly |
| Style Settings | Theme controls | Minimal options | Avoid extreme palettes | No content loss |
| Kanban | Visual pipelines | Project, content, medicine boards | Use Markdown boards | Boards remain readable lists |
| Omnisearch | Fast retrieval | Search concepts, cases, ideas | Index attachments if desired | Core search still works |
| Text Extractor | OCR support | Screenshots and PDFs | Manual OCR, no script reliance | Files still exist |
| Obsidian Git | Vault sync | Local commits and pull/push | See [[99_System/git/Obsidian_Git_Setup]] | Use Git externally |

Do not install plugin binaries into this repository.

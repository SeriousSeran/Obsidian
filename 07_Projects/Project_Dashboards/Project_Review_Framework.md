# Project Review Framework

> A simple system for weekly review of active projects.

## Review Steps

### 1. Status Check
- **Is this project still active?** Does it align with current goals?
- **Is it stuck?** What is the main blocker?

### 2. Next Actions
- **What is the immediate next step?** Ensure every active project has at least one clear, actionable task.
- **Who is responsible?** (Usually you, but clarify if waiting on someone else).

### 3. Decisions & Adjustments
- **Are there new decisions to record?**
- **Do the milestones need updating?**

### 4. Cleanup
- **Archive completed tasks.**
- **Move completed projects to the archive.**

## Rhythm

Perform this review weekly, ideally on Sunday or Monday morning, before starting new work. Use the [[07_Projects/Project_Dashboards/Active_Project_Dashboard]] as your guide.

# OSCE Image Recognition Product

## Product concept

A visual recognition product for OSCE/viva preparation.

Instead of vague image captions, each image-based card should identify the image, explain what matters, and teach the clinical reasoning.

## Problem

Many image-based flashcards fail because the back says only vague text like: this is an X-ray, discuss management.

A good card should behave like an intelligent clinical examiner.

## Ideal card back

For every image:

1. Identify the image.
2. State the likely diagnosis.
3. Mention the key visible signs.
4. Explain why those signs matter.
5. State immediate danger.
6. Give the exam/viva answer.
7. Add management priorities.
8. Add traps and differentials.
9. Add a memory hook if useful.

## Example structure

Image type: Chest X-ray.

Recognition: features suggestive of pulmonary edema in the correct clinical setting.

Visible clues: bilateral perihilar opacities, possible cardiomegaly, basal interstitial or alveolar shadowing.

Clinical meaning: acute decompensated heart failure may be present.

Immediate concerns: hypoxia, respiratory fatigue, cardiogenic shock, ACS or arrhythmia as trigger.

Viva answer: assess ABCDE, sit up, oxygen if hypoxic, monitors, ECG, IV access, treat pulmonary edema, search for precipitating cause.

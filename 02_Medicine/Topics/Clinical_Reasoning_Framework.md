# Clinical Reasoning Framework

> A reusable structure for moving from symptom to safe next step.

## First Pass

1. What is the main problem?
2. Is the patient unstable?
3. What dangerous diagnoses must not be missed?
4. What common diagnoses fit best?
5. What data would change management today?

## Problem Representation

Use one sentence:

> Age/context + key symptom + time course + important positives/negatives + risk factors.

## Differential Buckets

| Bucket | Use |
|---|---|
| Common | Most likely explanations |
| Dangerous | Time-critical diagnoses |
| Treatable | Diagnoses where early action helps |
| Contextual | Patient-specific risks and constraints |

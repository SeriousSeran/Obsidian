# Project Evaluation Framework

> A simple framework to evaluate if a project is worth starting.

## Evaluation Criteria

1. **Impact vs Effort**: High impact, low effort projects should be prioritized.
2. **Alignment**: Does this project align with my current goals and values?
3. **Risk**: What is the risk of failure? What is the cost of failure?
4. **Learning**: What new skills or knowledge will I gain from this project?
5. **Next Action**: What is the immediate next physical action required to start?

## Decision Matrix

| Project | Impact | Effort | Alignment | Risk | Decision (Go/No-Go) | Next Action |
|---|---|---|---|---|---|---|

## Notes

- Be honest about the effort required.
- Do not start a project without a clear next action.
